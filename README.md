## 📝 Engineering Principles

<div align="center">

| 💡 Principle | 🔍 What It Means to Me |
|---|---|
| **Clean Architecture** | Separation of concerns isn't optional — it's the foundation |
| **Performance by Default** | Every millisecond matters. Lazy load, memoize, optimize |
| **Type Safety First** | TypeScript strict mode, always. No `any`, no surprises |
| **Test What Matters** | Unit + integration tests on business logic — not just coverage % |
| **Ship, Then Improve** | Working software over perfect software. Then iterate |
| **Document as You Go** | Future you deserves clear comments and a helpful README |
